总分：S_total = 0.4*S_seg + 0.4*S_cls + 0.2*S_time
分割 S_seg：long/trans 各 50%；每视图 0.6*plaque + 0.4*vessel，且
S_{v,plaque} = (DSC_{v,plaque} + NSD_{v,plaque})/2，S_{v,vessel} = (DSC_{v,vessel} + NSD_{v,vessel})/2
分类 S_cls：按类别的 macro-F1：S_cls = (1/N) * Σ F1_i（二分类时就是两类 F1 的平均）
时间 S_time：相对 T_baseline 做阈值截断（2/3*T_baseline 和 2*T_baseline，分3种case确定 T_min/T_max），再 min-max：
S_time = (T_max - T_user)/(T_max - T_min) * 100
